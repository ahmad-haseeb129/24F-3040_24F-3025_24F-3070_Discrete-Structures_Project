# UNIDISC ENGINE - FAST University
 
## Overview
UNIDISC ENGINE is a console-based application designed to manage university data, including students, courses, faculty, and rooms. The application offers two modes: Interactive Mode for user interaction and Test Mode for running predefined tests.
## Project Structure
- `main.cpp`: The main entry point of the application. It contains the main menu loop that allows users to choose between Interactive Mode, Test Mode, or exit the program.
## Modules
- **Data Structures**: Defines the entities used in the application.
- **CLI Interface**: Manages user input and output for Interactive Mode.
- **Testing Framework**: Contains tests to ensure the application functions correctly.
- **Course Management**: Handles operations related to courses, including prerequisites and enrollment.
- **Student Management**: Manages student data, including completed and current courses.
- **Faculty Management**: Manages faculty data and their assigned courses.
- **Room Management**: Manages room data and their capacities.
- **Sets and Maps Utilities**: Provides utility functions for handling sets and maps.
 
## How to Use
1. Compile the project using a C++ compiler.
2. Run the compiled executable.
3. Choose between Interactive Mode or Test Mode from the main menu.
4. In Interactive Mode, follow the prompts to manage university data.
5. In Test Mode, the application will run predefined tests and display the results.
6. To exit the application, select the exit option from the main menu.
## Sample Data
The application includes a function to add sample data for testing purposes. This data includes sample students, courses, faculty, and rooms to demonstrate the application's functionality.
## Conclusion
UNIDISC ENGINE is a simple yet effective tool for managing university data. Its modular structure allows for easy expansion and maintenance. Users can interact with the system through a command-line interface, making it accessible and straightforward to use.
